## 六、实战：多模型路由

```python
from openai import OpenAI
import anthropic

class ModelRouter:
    def __init__(self):
        self.deepseek = OpenAI(
            base_url="https://api.deepseek.com",
            api_key="xxx"
        )
        self.claude = anthropic.Anthropic(api_key="xxx")
    
    def route(self, prompt, task_type):
        if task_type == "code":
            return self.claude.messages.create(
                model="claude-3-5-sonnet-20260620",
                messages=[{"role": "user", "content": prompt}]
            )
        elif task_type == "simple":
            return self.deepseek.chat.completions.create(
                model="deepseek-chat",
                messages=[{"role": "user", "content": prompt}]
            )
        else:
            return self.claude.messages.create(
                model="claude-3-5-sonnet-20260620",
                messages=[{"role": "user", "content": prompt}]
            )

# 使用
router = ModelRouter()
response = router.route("写一个快速排序", task_type="code")
```

## 八、常见问题

### Q1：开源模型能商用吗？

**答**：大部分可以，但要注意：
- LLaMA-3：可商用
- Qwen2.5：可商用
- ChatGLM3：可商用
- Yi：需要申请

### Q2：API 会不会泄露数据？

**答**：
- 商业 API（GPT/Claude）：可能用于训练
- 国内 API（DeepSeek）：承诺不训练
- 敏感数据：本地部署

### Q3：如何选择上下文长度？

**答**：
- 普通对话：8K 足够
- 文档分析：32K+
- 长篇小说：128K
