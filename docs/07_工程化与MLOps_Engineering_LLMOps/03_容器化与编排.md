## 一、为什么用 Docker

**优势**：
- ✅ 环境一致（开发=生产）
- ✅ 快速部署
- ✅ 资源隔离
- ✅ 易于扩展

## 二、Docker 基础

### 1. 安装

```bash
# Ubuntu
curl -fsSL https://get.docker.com | sh

# macOS
brew install --cask docker
```

### 2. Hello World

```bash
docker run hello-world
```

### 3. 常用命令

```bash
docker ps              # 查看运行容器
docker images          # 查看镜像
docker build -t myapp . # 构建镜像
docker run -d -p 8000:8000 myapp  # 运行容器
docker logs <container_id>        # 查看日志
docker stop <container_id>        # 停止容器
```

## 三、Dockerfile 编写

### 1. Python 应用

```dockerfile
FROM python:3.11-slim

WORKDIR /app

# 复制依赖
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# 复制代码
COPY . .

# 暴露端口
EXPOSE 8000

# 启动命令
CMD ["python", "main.py"]
```

### 2. 多阶段构建（减小体积）

```dockerfile
# 构建阶段
FROM python:3.11 as builder

WORKDIR /app
COPY requirements.txt .
RUN pip install --user --no-cache-dir -r requirements.txt

# 运行阶段
FROM python:3.11-slim

WORKDIR /app
COPY --from=builder /root/.local /root/.local
COPY --from=builder /app .

ENV PATH=/root/.local/bin:$PATH

CMD ["python", "main.py"]
```

## 四、Docker Compose

### 1. 配置文件

```yaml
version: '3.8'

services:
  api:
    build: .
    ports:
      - "8000:8000"
    environment:
      - DATABASE_URL=postgresql://user:pass@db:5432/mydb
    depends_on:
      - db
  
  db:
    image: postgres:15
    environment:
      - POSTGRES_USER=user
      - POSTGRES_PASSWORD=pass
      - POSTGRES_DB=mydb
    volumes:
      - postgres_data:/var/lib/postgresql/data

volumes:
  postgres_data:
```

### 2. 启动

```bash
docker compose up -d
docker compose down
docker compose logs -f
```

## 五、LLM 应用部署

### 1. Ollama + FastAPI

```dockerfile
FROM ollama/ollama

RUN ollama pull llama3

COPY app.py .

CMD ["python", "app.py"]
```

### 2. vLLM 部署

```yaml
version: '3.8'

services:
  vllm:
    image: vllm/vllm-openai
    ports:
      - "8000:8000"
    volumes:
      - ~/.cache/huggingface:/root/.cache/huggingface
    environment:
      - HUGGING_FACE_HUB_TOKEN=your_token
    command: >
      --model meta-llama/Meta-Llama-3-8B-Instruct
      --port 8000
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: 1
              capabilities: [gpu]
```

## 六、优化技巧

### 1. 减小镜像体积

```dockerfile
# 用 slim 基础镜像
FROM python:3.11-slim

# 清理缓存
RUN apt-get clean && rm -rf /var/cache/apt/*

# 多阶段构建
FROM ... as builder
...
FROM ... as runner
COPY --from=builder ...
```

### 2. 加速构建

```dockerfile
# 先复制依赖文件
COPY requirements.txt .
RUN pip install -r requirements.txt

# 再复制代码（变化频繁）
COPY . .
```

### 3. .dockerignore

```
__pycache__/
*.pyc
.git/
.env
venv/
```

## 七、常见问题

### Q1：GPU 支持？

**答**：
```bash
# 需要 NVIDIA Container Toolkit
docker run --gpus all my-llm-app
```

### Q2：数据持久化？

**答**：
```yaml
volumes:
  - ./data:/app/data  # 绑定挂载
  - myvolume:/data    # 命名卷
```

### Q3：如何调试？

**答**：
```bash
# 进入容器
docker exec -it <container_id> bash

# 查看日志
docker logs -f <container_id>

# 查看资源使用
docker stats
```
