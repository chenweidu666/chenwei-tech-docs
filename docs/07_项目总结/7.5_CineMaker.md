# 7.5. CineMaker · 多Agent短剧AI生产平台

> **时间**：2025.12 – 至今｜**性质**：个人项目

## 7.5.1. 项目背景
- 短剧内容生产人力成本高
- 目标：1人+1小时+约20元完成单集

## 7.5.2. 多Agent架构
- 资源库Agent → 剧本Agent → 分镜Agent → 生图Agent → 合成Agent
- 端到端全自动工作流

## 7.5.3. 核心技术
- Go/Gin 后端 + Vue3/TS 前端
- LangChain 编排
- 火山引擎（视频生成）
- FFmpeg（合成）

## 7.5.4. 角色一致性方案
- 参考图驱动（三视图+场景空镜）
- 三段式分镜描述（首帧/动作/尾帧）
- 解决跨镜头漂移问题

## 7.5.5. 核心成果
- 13镜头/约90秒短剧：1小时，约20元
- 首轮成片率XX%
