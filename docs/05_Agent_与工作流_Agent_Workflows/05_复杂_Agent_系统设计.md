## 一、什么是 AppAgent

**AppAgent**：用大模型控制手机 App。

**核心思想**：截图 + 识别 + 操作。

```
屏幕截图 → LLM 分析 → 生成操作 → ADB 执行
```

---

## 二、架构

```
┌─────────────┐
│   LLM       │ ← 大脑（决策）
└──────┬──────┘
       │
┌──────┴──────┐
│ AppAgent    │ ← 协调器
└──────┬──────┘
       │
┌──────┴──────┐
│   ADB       │ ← 执行（点击/滑动/输入）
└──────┬──────┘
       │
┌──────┴──────┐
│  Android    │ ← 手机/模拟器
└─────────────┘
```

---

## 三、快速开始

### 1. 安装依赖

```bash
pip install appagent
adb install appagent.apk
```

### 2. 连接设备

```bash
adb devices
# 确保设备已连接
```

### 3. 运行示例

```python
from appagent import AppAgent

agent = AppAgent(
    model="gpt-4-vision",
    device="emulator-5554"
)

# 打开微信
agent.run("打开微信，给张三发消息：你好")
```

## 四、应用场景

### 1. 自动化测试

```python
# 测试登录流程
agent.run("打开 App，输入用户名密码，点击登录")
```

### 2. 数据采集

```python
# 抓取小红书笔记
agent.run("打开小红书，搜索 AI，收集前 10 篇笔记标题")
```

### 3. 批量操作

```python
# 批量点赞
agent.run("打开抖音，给前 5 个视频点赞")
```

## 五、限制与挑战

### 当前限制

- ❌ 速度较慢（2-5 秒/步）
- ❌ 复杂流程易出错
- ❌ 需要屏幕截图

### 优化方向

- ✅ 缓存常用操作
- ✅ 增加错误恢复
- ✅ 多设备并行
