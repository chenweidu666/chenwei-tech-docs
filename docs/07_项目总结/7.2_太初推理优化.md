# 7.2. 太初 · 大模型推理优化系统

> **时间**：2024.07 – 2025.07｜**公司**：太初电子科技（上海）

## 7.2.1. 项目背景
- 国产GPU集群（8卡），推理性能不足
- 成本高、延迟大、可用性低

## 7.2.2. 优化方案
- vLLM + PagedAttention
- INT8/INT4混合量化（敏感层保FP16）
- 8卡张量并行 + 流水线并行
- CI/CD一键部署

## 7.2.3. 核心成果
- 吞吐↑2-3×
- 显存-50%+
- P95延迟 100-150ms，80+ QPS
- 可用性 99.9%+
- GPU成本-30~40%
