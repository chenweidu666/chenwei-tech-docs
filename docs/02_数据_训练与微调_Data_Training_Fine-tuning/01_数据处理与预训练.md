## 三、数据清洗流程

### 1. 去重

```python
from datasketch import MinHash, MinHashLSH

def deduplicate(texts, threshold=0.8):
    """使用 MinHash 去重"""
    lsh = MinHashLSH(threshold=threshold)
    
    for i, text in enumerate(texts):
        m = MinHash()
        for word in text.split():
            m.update(word.encode('utf8'))
        lsh.insert(f"doc_{i}", m)
    
    # 返回唯一文档
    unique = []
    seen = set()
    for i, text in enumerate(texts):
        if f"doc_{i}" not in seen:
            unique.append(text)
            seen.add(f"doc_{i}")
    
    return unique
```

### 2. 质量过滤

```python
def filter_quality(text):
    """过滤低质量文本"""
    # 太短
    if len(text) < 100:
        return False
    
    # 特殊字符太多
    special_ratio = sum(not c.isalnum() for c in text) / len(text)
    if special_ratio > 0.3:
        return False
    
    # 包含敏感词
    if any(word in text for word in ['广告', '加微信', '点击领取']):
        return False
    
    return True
```

### 3. 格式标准化

```python
import re

def normalize_text(text):
    """标准化文本格式"""
    # 移除多余空格
    text = re.sub(r'\s+', ' ', text)
    
    # 统一标点
    text = text.replace('"', '"').replace('"', '"')
    
    # 移除 URL
    text = re.sub(r'http[s]?://\S+', '', text)
    
    return text.strip()
```

### 2. 数据转换脚本

```python
import json

def convert_to_alpaca(data):
    """转换为 Alpaca 格式"""
    alpaca_data = []
    
    for item in data:
        messages = item['messages']
        
        # 提取指令和输入
        instruction = messages[0]['content']
        input_text = messages[1]['content'] if len(messages) > 1 else ""
        output = messages[-1]['content']
        
        alpaca_data.append({
            "instruction": instruction,
            "input": input_text,
            "output": output
        })
    
    return alpaca_data

# 使用
with open('data.json') as f:
    data = json.load(f)

alpaca_format = convert_to_alpaca(data)

with open('alpaca_data.json', 'w') as f:
    json.dump(alpaca_format, f, ensure_ascii=False, indent=2)
```

## 六、常见问题

### Q1：需要多少数据？

**答**：
- 全量微调：1000+ 条
- LoRA 微调：100+ 条
- 少样本学习：10 条即可

### Q2：数据质量如何评估？

**答**：
- 人工抽检（100 条）
- 自动指标（困惑度）
- A/B 测试（上线对比）

### Q3：中文数据比例多少合适？

**答**：
- 通用模型：30-50% 中文
- 中文专用：70%+ 中文
- 保持多样性，避免过拟合
